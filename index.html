  • 내용 : 세션을 사용한 로그인 기능 수업 이후 수업내용을 바탕으로 JSP 구조의 게시판 구현
기능
  • 로그인/로그아웃
    • 세션을 사용한 로그인/로그아웃
    • 요청값 유효성 검사 후 null일 경우 메세지 출력
    • 로그인 시 아이디, 비밀번호 일치 확인 검사 후 일치하지 않는 경우 로그인 실패, 해당 페이지에 메세지 출력
  • 회원가입
    • 중복 아이디 검사 후 중복된 아이디 값이 요청되면 가입 실패, 해당 페이지에 메세지 출력
  • 카테고리, 게시판 CRUD
    • 카테고리는 관리자(admin)만 생성, 수정, 삭제 할 수 있도록 카테고리 관리 버튼 유무 분기
    • 게시글이 있는 카테고리는 수정, 삭제 불가
    • 게시글은 로그인 회원만 작성 가능하도록 글작성 버튼 유무 분기
    • 게시글은 작성자만 수정, 삭제 가능하도록 작성자 id와 세션에 저장된 id값 비교하여 수정, 삭제 버튼 유무 분기
    • 댓글이 있는 게시글은 외래키 CASCADE 사용하여 댓글 과 함께 삭제
    • 전체, 카테고리별 게시글 조회
    • 게시글 리스트 조회 시 페이징
